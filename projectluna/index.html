F.A.Q
When will this release? Sometime between tomorrow and never.
Why did you use a 2d framework for a fpv game? When I began this project I only knew Lua so it felt less daunting than learning a proper 3d engine + a new language.
How did you make a "3d" game using a 2d framework? With the dumbest implementation of raycasting you've ever seen, I call it A.I.R.C, An Idiot's Ray Caster.
Will this game be like [MS-DOS era RPG]? While I've definitely taken inspiration from those games I hope to make a game that's less clunky to play than say Ultima UW or TES Arena, though those games will probably still be better in all other ways.
Doesn't this seem a little overambitious for a single person? Most certainly, hence my answer to the first question.
